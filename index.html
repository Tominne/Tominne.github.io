Research Topic

What is the difference between 'margin' 'border' and 'padding'?

All three properties are of the CSS box model, that when linked with an HTML file influence the style of elements.

  1. 'Margin' defines the space outside one element and the elements around it without effecting the element itself!
    margin: 10px 20px 30px 40px; will effect each side of the element independently, starting from the top, then going right like a clock hand.


  2. 'Border' defines the look of the border edge layer, of an element, visible or not depending on style attributes given (solid, hidden color, dashed, radius, etc).


  3. 'Padding' will add to the size of an element, giving it some roomy breathing space inside the element around the text, image, or other content.
